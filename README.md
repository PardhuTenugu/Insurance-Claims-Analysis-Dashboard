# 🛡️ Insurance Claims Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Insurance Claims Analysis Dashboard** built using **Power BI**. The dashboard provides insights into insurance claims, customer demographics, and business performance through interactive visualizations and KPIs.

It helps analyze key metrics, identify trends, and support data-driven decision-making.

---

## 🎯 Objective

The main objective of this project is to:

- Analyze insurance claims and customer data.
- Monitor key business performance metrics.
- Identify claim patterns and customer trends.
- Create an interactive dashboard for business insights.
